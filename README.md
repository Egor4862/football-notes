# Футбольные заметки

Простое веб-приложение для создания и хранения заметок о футболе.

## Что использовалось в проекте:
* Python, FastAPI, Uvicorn
* SQLite (sqlite3)
* HTML, CSS, JavaScript (jQuery для запросов, Bootstrap для сетки)

## Что умеет приложение:
* Создавать новые заметки (тема, дата, категория).
* Сохранять всё в локальную базу данных (`lr3.db`).
* Показывать список всех заметок при загрузке страницы.
* Фильтровать заметки по категориям (Трансферы, Матчи и т.д.).
* Очищать всю базу данных одной кнопкой.

## Как запустить у себя:

1. Установить библиотеки:
```bash
pip install -r requirements.txt
```
2. Запустить сервер:
```bash
python server.py
```
3. Открыть браузер и перейти по адресу:

http://localhost:12345/
